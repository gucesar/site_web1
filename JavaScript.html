JavaScript

O que é JavaScript?

JavaScript é uma linguagem de programação de alto nível e interpretada que é amplamente utilizada para adicionar interatividade e dinamismo a sites web. É uma das linguagens fundamentais da web, ao lado de HTML e CSS, e é suportada por todos os navegadores modernos.

Para que Serve?

JavaScript serve para:

  • Interatividade: Permite adicionar interatividade às páginas web, como formulários dinâmicos, menus suspensos e validação de dados.
  • Manipulação do DOM: Facilita a manipulação e atualização do conteúdo HTML e CSS da página de forma dinâmica, sem a necessidade de recarregar a página.
  • Comunicação com Servidores: Permite a comunicação assíncrona com servidores através de AJAX, possibilitando atualizações de conteúdo sem recarregar a página.
  • Desenvolvimento de Aplicações Web: Utilizado para criar aplicações web completas, desde jogos até ferramentas de produtividade e plataformas interativas.

Como Funciona?

  • Inclusão no HTML: O JavaScript pode ser incluído em uma página HTML usando a tag <script>. Pode ser colocado no <head> ou no final do <body>.
  • Execução no Navegador: O código JavaScript é executado pelo navegador do usuário, permitindo a interação direta com o DOM e a manipulação de elementos da página.
  • Eventos: JavaScript responde a eventos do usuário, como cliques, movimentos do mouse e entrada de dados, para executar funções específicas.
  • AJAX e Fetch: Permite a comunicação assíncrona com servidores para obter ou enviar dados sem recarregar a página, utilizando tecnologias como AJAX e Fetch API.

Exemplo de Código JavaScript

Exemplo Simples:


<script>
    function mostrarMensagem() {
        alert('Olá, Mundo!');
    }
</script>

<button onclick="mostrarMensagem()">Clique aqui</button>

Este código cria uma função "mostrarMensagem" que exibe uma mensagem de alerta quando um botão é clicado.

Exemplo com AJAX:


<script>
    function carregarDados() {
        fetch('https://api.exemplo.com/dados')
            .then(response => response.json())
            .then(data => {
                console.log(data);
                document.getElementById('resultado').innerText = JSON.stringify(data);
            })
            .catch(error => console.error('Erro:', error));
    }
</script>

<button onclick="carregarDados()">Carregar Dados</button>
<div id="resultado"></div>

Este código usa a Fetch API para obter dados de um servidor e exibi-los na página sem recarregar.

Principais Conceitos de JavaScript

Alguns conceitos importantes incluem:

  • Variáveis e Tipos de Dados: JavaScript suporta vários tipos de dados, incluindo números, strings, objetos e arrays, e permite a declaração de variáveis usando var, let ou const.
  • Funções: Funções são blocos de código reutilizáveis que realizam uma tarefa específica e podem ser declaradas de várias formas.
  • Objetos e Arrays: JavaScript permite a criação e manipulação de objetos e arrays para armazenar e organizar dados de maneira eficiente.
  • Assíncrono e Promises: JavaScript suporta programação assíncrona, permitindo que funções sejam executadas sem bloquear o restante do código. Promises e async/await são usados para lidar com operações assíncronas.
  • Eventos e Manipulação do DOM: JavaScript pode ouvir e responder a eventos do usuário e manipular o DOM para alterar o conteúdo da página dinamicamente.
