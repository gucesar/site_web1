DNS (Domain Name System)

O que é DNS?

DNS (Domain Name System) é um sistema de nomenclatura que traduz nomes de domínio em endereços IP. Ele permite que os usuários acessem websites usando nomes legíveis em vez de endereços numéricos, facilitando a navegação na internet.

Para que Serve?

O DNS serve para:

  • **Resolução de Nomes:** Converte nomes de domínio (como www.exemplo.com) em endereços IP (como 192.0.2.1), permitindo a localização de servidores na internet.
  • **Gerenciamento de Domínios:** Permite a configuração de diversos tipos de registros (como A, AAAA, CNAME, MX) que controlam como os domínios são resolvidos.
  • **Distribuição de Carga:** Utiliza técnicas como DNS Round Robin para distribuir o tráfego entre múltiplos servidores, melhorando a performance e a disponibilidade.
  • **Segurança:** Proporciona recursos de segurança como DNSSEC (Domain Name System Security Extensions), que adiciona uma camada de autenticação às respostas DNS.

Como Funciona?

  • **Hierarquia de Domínios:** O DNS é organizado em uma estrutura hierárquica, começando pelos domínios de nível superior (TLDs) como .com, .org, e .net.
  • **Servidores DNS:** Existem diferentes tipos de servidores DNS, incluindo servidores autoritativos (que armazenam os registros de um domínio) e servidores recursivos (que buscam os registros necessários para resolver uma consulta).
  • **Processo de Resolução:** Quando um usuário digita um nome de domínio, uma consulta é enviada a um servidor DNS recursivo, que busca a informação em cache ou realiza novas consultas até encontrar o endereço IP correspondente.
  • **Cache de DNS:** Para otimizar o desempenho, os resultados das consultas DNS são armazenados em cache por um período determinado, reduzindo o tempo necessário para resolver nomes de domínio repetidamente.

Exemplo de Configuração DNS

Exemplo de Registro DNS:


; Registro A
exemplo.com.   IN  A   192.0.2.1

; Registro CNAME
www.exemplo.com.  IN  CNAME  exemplo.com.

; Registro MX
exemplo.com.   IN  MX  10 mail.exemplo.com.

Principais Conceitos do DNS

Alguns conceitos importantes incluem:

  • **Domínios:** A estrutura que organiza a hierarquia dos nomes na internet, sendo composta por subdomínios e domínios de nível superior.
  • **Registros DNS:** Tipos de informações armazenadas no DNS, como registros A (endereços IPv4), AAAA (endereços IPv6), CNAME (nomes canônicos) e MX (servidores de email).
  • **TTL (Time To Live):** O tempo que um registro DNS permanece em cache antes de ser considerado obsoleto e precisar ser consultado novamente.
  • **DNSSEC:** Extensões de segurança que ajudam a proteger o DNS contra ataques como spoofing e cache poisoning.
