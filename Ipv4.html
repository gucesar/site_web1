IPv4 (Protocolo de Internet versão 4)

O que é o IPv4?

O IPv4, ou Protocolo de Internet versão 4, é uma versão do protocolo IP (Protocolo de Internet) utilizado para endereçamento e roteamento de pacotes em redes. Desenvolvido na década de 1980, o IPv4 é a versão mais amplamente utilizada do protocolo IP, apesar de estar sendo gradualmente substituído pelo IPv6 devido ao esgotamento dos endereços disponíveis.

Para que Serve?

O IPv4 serve para:

  • **Endereçamento:** Fornece um esquema de endereçamento único para identificar dispositivos em uma rede.
  • **Roteamento de Pacotes:** Permite o roteamento de pacotes de dados entre dispositivos em redes locais e na Internet.
  • **Comunicação de Dados:** Facilita a comunicação de dados entre computadores e outros dispositivos em uma rede.
  • **Identificação de Dispositivos:** Cada dispositivo em uma rede é atribuído um endereço IPv4 único, permitindo que se comunique com outros dispositivos.
