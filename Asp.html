ASP (Active Server Pages)

O que é ASP?

ASP (Active Server Pages) é uma tecnologia da Microsoft para criar páginas web dinâmicas e interativas. Foi introduzida em 1996 como parte do Microsoft Internet Information Services (IIS). ASP permite que o código seja executado no servidor, gerando HTML dinâmico que é enviado ao navegador do cliente.

Para que Serve?

ASP serve para:

  • Geração de Conteúdo Dinâmico: Permite a criação de páginas web dinâmicas que podem gerar conteúdo com base em entradas do usuário, consultas a banco de dados, ou outras fontes de dados.
  • Integração com Bancos de Dados: Facilita a conexão e a manipulação de bancos de dados, permitindo a recuperação e a atualização de informações armazenadas.
  • Criação de Aplicações Web: Utilizado para desenvolver aplicações web completas que requerem interação com o servidor, como sistemas de gerenciamento de conteúdo e e-commerce.
  • Manipulação de Dados do Usuário: Permite a coleta e processamento de dados enviados pelos usuários através de formulários e outras interações.

Como Funciona?

  • Processamento no Servidor: O código ASP é processado no servidor web antes de ser enviado ao navegador. O servidor executa o código, gera o HTML dinâmico e o envia ao cliente.
  • Instruções ASP: O código ASP é escrito dentro de blocos especiais no arquivo .asp. Esses blocos são delimitados por `<%` e `%>`, onde o código é executado no servidor.
  • Integração com HTML: ASP pode ser misturado com HTML, permitindo a geração de conteúdo dinâmico dentro das tags HTML. Isso facilita a inclusão de dados gerados dinamicamente na estrutura da página.
  • Conexão com Bancos de Dados: ASP pode utilizar ADO (ActiveX Data Objects) ou outras tecnologias para se conectar a bancos de dados, executar consultas SQL e manipular dados.

Exemplo de Código ASP

Exemplo Simples:


<% 
    Dim nome
    nome = "Mundo"
%>

<html>
<body>
    <h1>Olá, <%= nome %>!</h1>
</body>
</html>

Este código define uma variável "nome" no código ASP e a exibe na página HTML gerada.

Exemplo com Banco de Dados:


<% 
    Dim conn, rs, sql
    Set conn = Server.CreateObject("ADODB.Connection")
    conn.Open "DSN=nomeDoDSN;UID=usuario;PWD=senha;"
    
    sql = "SELECT * FROM tabela"
    Set rs = conn.Execute(sql)
    
    Response.Write "<table>"
    Do While Not rs.EOF
        Response.Write "<tr><td>" & rs("campo1") & "</td><td>" & rs("campo2") & "</td></tr>"
        rs.MoveNext
    Loop
    Response.Write "</table>"
    
    rs.Close
    conn.Close
    Set rs = Nothing
    Set conn = Nothing
%>

Este código se conecta a um banco de dados, executa uma consulta SQL e exibe os resultados em uma tabela HTML.

Principais Conceitos de ASP

Alguns conceitos importantes incluem:

  • Objetos ASP: ASP utiliza objetos incorporados como `Request`, `Response`, `Session` e `Server` para gerenciar informações e controlar o fluxo de dados.
  • Variáveis e Tipos de Dados: ASP suporta variáveis que podem armazenar diferentes tipos de dados, incluindo strings, números e objetos.
  • Incluindo Código e Arquivos: ASP permite a inclusão de arquivos externos e reutilização de código com a diretiva `Include`.
  • Tratamento de Erros: O tratamento de erros em ASP pode ser feito com a instrução `On Error Resume Next` para capturar e gerenciar erros de forma controlada.
