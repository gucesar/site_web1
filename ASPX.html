ASPX (Active Server Pages Extended)

O que é ASPX?

ASPX é uma extensão de arquivo usada pelo ASP.NET, uma framework de desenvolvimento web da Microsoft. ASPX permite a criação de páginas web dinâmicas e interativas, combinando código C# ou VB.NET com HTML, CSS e JavaScript.

Para que Serve?

O ASPX serve para:

  • **Desenvolvimento de Aplicações Web:** Facilita a criação de aplicações web complexas, com suporte a eventos, controles e componentes.
  • **Geração Dinâmica de Conteúdo:** Permite gerar conteúdo HTML dinâmico com base em interações do usuário e dados de servidores.
  • **Integração com Banco de Dados:** Facilita a conexão e manipulação de dados em bancos de dados através de ADO.NET e Entity Framework.
  • **Suporte a Controle de Estado:** Permite manter o estado de informações entre requisições, utilizando ViewState e Session.

Como Funciona?

  • **Modelo de Página:** Uma página ASPX é composta por um arquivo de interface (ASPX) e um arquivo de código-behind (C# ou VB.NET) que contém a lógica da aplicação.
  • **Ciclo de Vida da Página:** O ASP.NET possui um ciclo de vida de página bem definido, que inclui etapas como inicialização, carregamento, manipulação de eventos e renderização.
  • **Controles de Servidor:** ASPX utiliza controles de servidor que permitem a criação de elementos dinâmicos, como botões, grids e formulários, que podem ser manipulados no código.
  • **Tratamento de Eventos:** O código pode reagir a eventos como cliques de botão e mudanças em campos de entrada, permitindo interatividade nas páginas.

Exemplo de Código ASPX

Exemplo Simples de ASPX:


<%@ Page Language="C#" AutoEventWireup="true" CodeFile="Exemplo.aspx.cs" Inherits="Exemplo">
<html>
<head><title>Exemplo de ASPX</title></head>
<body>
    <form id="form1" runat="server">
        <div>
            <h1>Bem-vindo, <asp:Label ID="lblNome" runat="server" />!</h1>
            <asp:TextBox ID="txtNome" runat="server" />
            <asp:Button ID="btnEnviar" runat="server" Text="Enviar" OnClick="btnEnviar_Click" />
        </div>
    </form>
</body>
</html>

Código-behind (Exemplo.aspx.cs):


using System;

public partial class Exemplo : System.Web.UI.Page
{
    protected void btnEnviar_Click(object sender, EventArgs e)
    {
        lblNome.Text = txtNome.Text;
    }
}

Principais Conceitos do ASPX

Alguns conceitos importantes incluem:

  • **Modelo de Eventos:** O ASP.NET utiliza um modelo baseado em eventos, onde os controles de servidor geram eventos que podem ser manipulados no código-behind.
  • **State Management:** Técnicas para gerenciar o estado da aplicação, como ViewState, Session, Cookies e Query Strings.
  • **Master Pages:** Permitem a criação de layouts consistentes em toda a aplicação, onde uma página mestre define a estrutura comum e páginas filhas herdam esse layout.
  • **Web Forms:** ASPX é usado principalmente com Web Forms, um modelo de programação que facilita o desenvolvimento de interfaces de usuário ricas.
