CSS (Cascading Style Sheets)

O que é CSS?

CSS (Cascading Style Sheets), ou Folhas de Estilo em Cascata, é uma linguagem de estilo utilizada para descrever a apresentação de documentos escritos em HTML ou XML (incluindo suas variantes como SVG e XHTML). O CSS define como os elementos devem ser exibidos em tela, papel ou em outros meios.

Para que Serve?

O CSS serve para:

  • **Controle de Layout:** Permite o controle preciso do layout e da posição dos elementos em uma página, tornando possível criar designs complexos e responsivos.
  • **Estilização:** Define a aparência visual dos elementos, incluindo cores, fontes, tamanhos e espaçamentos.
  • **Separação de Conteúdo e Apresentação:** Permite separar o conteúdo HTML da sua apresentação visual, facilitando a manutenção e a reutilização do código.
  • **Responsividade:** Facilita a criação de layouts que se adaptam a diferentes tamanhos de tela e dispositivos, como computadores, tablets e smartphones.

Como Funciona?

  • **Seletores:** O CSS utiliza seletores para aplicar estilos a elementos HTML. Seletores podem ser baseados em tags, classes, IDs ou atributos.
  • **Propriedades e Valores:** Cada seletor pode ter uma ou mais propriedades com valores que definem o estilo do elemento. Por exemplo, `color: red;` define a cor do texto.
  • **Cascata e Herança:** O CSS aplica estilos com base na "cascata", onde regras mais específicas ou com maior importância sobrepõem regras mais gerais. A herança permite que estilos aplicados a elementos pais sejam passados para seus filhos.
  • **Folhas de Estilo Internas e Externas:** O CSS pode ser incorporado diretamente em uma página HTML (CSS interno) ou em arquivos separados (CSS externo) que são vinculados à página.

Exemplo de Código CSS

Exemplo de CSS Interno:


<style>
    body {
        font-family: Arial, sans-serif;
        background-color: #f0f0f0;
    }
    h1 {
        color: #333;
        text-align: center;
    }
    p {
        color: #666;
        line-height: 1.6;
    }
</style>

Exemplo de CSS Externo:


<link rel="stylesheet" href="styles.css">

Em um arquivo CSS externo (`styles.css`):


body {
    font-family: Arial, sans-serif;
    background-color: #f0f0f0;
}
h1 {
    color: #333;
    text-align: center;
}
p {
    color: #666;
    line-height: 1.6;
}

Principais Conceitos de CSS

Alguns conceitos importantes incluem:

  • **Box Model:** Define como o conteúdo, padding, borda e margem dos elementos são calculados e exibidos.
  • **Flexbox:** Um modelo de layout que facilita a criação de layouts flexíveis e responsivos.
  • **Grid Layout:** Um sistema de layout que permite criar layouts bidimensionais complexos com linhas e colunas.
  • **Media Queries:** Permitem aplicar estilos diferentes com base nas características do dispositivo, como tamanho da tela e orientação.
