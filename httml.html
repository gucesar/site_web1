HTML (HyperText Markup Language)

O que é HTML?

HTML (HyperText Markup Language) é a linguagem de marcação padrão utilizada para criar páginas da web. Ela fornece a estrutura básica do conteúdo na web, utilizando uma combinação de elementos, tags e atributos para organizar e formatar o texto, imagens, links e outros conteúdos.

Para que Serve?

O HTML serve para:

  • **Estruturação de Conteúdo:** Define a estrutura de um documento web, organizando o conteúdo em seções, cabeçalhos, parágrafos, listas e muito mais.
  • **Exibição de Multimídia:** Permite a inclusão de imagens, vídeos e outros conteúdos multimídia em uma página web.
  • **Criação de Links:** Facilita a navegação entre páginas e sites por meio de links (hiperlinks).
  • **Formulários:** Suporta a criação de formulários interativos para coleta de dados do usuário.

Como Funciona?

  • **Tags e Elementos:** O HTML é composto por tags que definem diferentes elementos. Cada tag tem um significado e pode conter atributos para personalização.
  • **Estrutura de um Documento HTML:** Um documento HTML começa com a declaração do tipo do documento, seguido pela estrutura básica, incluindo , , e .
  • **Aninhamento de Elementos:** Os elementos HTML podem ser aninhados dentro de outros elementos, permitindo a criação de hierarquias e layouts complexos.
  • **Atributos:** As tags HTML podem ter atributos que modificam seu comportamento ou aparência, como class, id, style, etc.

Exemplo de Código HTML

Exemplo Simples de HTML:


<!DOCTYPE html>
<html lang="pt-BR">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Exemplo de HTML</title>
</head>
<body>
    <header>
        <h1>Bem-vindo ao Meu Site</h1>
    </header>

    <nav>
        <ul>
            <li><a href="#sobre">Sobre</a></li>
            <li><a href="#contato">Contato</a></li>
        </ul>
    </nav>

    <section id="sobre">
        <h2>Sobre Mim</h2>
        <p>Este é um exemplo de uma página HTML simples.</p>
    </section>

    <section id="contato">
        <h2>Contato</h2>
        <p>Você pode me contatar pelo email: exemplo@dominio.com</p>
    </section>

    <footer>
        <p>© 2024 Meu Site</p>
    </footer>
</body>
</html>

Principais Conceitos do HTML

Alguns conceitos importantes incluem:

  • **Tags Semânticas:** Tags como <header>, <footer>, <article> e <section> que ajudam a descrever o significado do conteúdo.
  • **Acessibilidade:** Práticas que garantem que o conteúdo seja acessível a todos os usuários, incluindo aqueles com deficiências.
  • **HTML5:** A versão mais recente do HTML, que introduz novas funcionalidades e elementos, como suporte a áudio e vídeo nativo.
  • **Validação de HTML:** Processo de verificar se o código HTML está correto e em conformidade com os padrões estabelecidos.
